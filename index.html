💻 Python Ambiental - Clase 1

Repositorio interactivo basado en JupyterLite.

Abrir notebook en el navegador

🧠 Instrucciones

  1. Espera unos segundos mientras carga el entorno JupyterLite.
  2. Si aparece el mensaje “Select Kernel”, elige:
    🐍 Python 3 (Pyodide).
  3. Ejecuta las celdas con Shift + Enter.
  4. Si el notebook no se abre automáticamente, usa el menú:
    File → Open from → notebooks/ejemplo.ipynb.
